◆ autoReserve()

bool SR830::autoReserve ( ) const

Performs the Auto Scale (Auto Gain) function.

The ARSV command performs the Auto Reserve function. This command is the same as pressing the [Auto Reserve] key. Auto Reserve may take some time. Check the command execution in progress bit in the Serial Poll Status Byte (bit 1) to determine when the function is finished.

Returns
True if quering is correct
