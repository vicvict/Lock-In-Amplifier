Detailed Description

Common class for all the LIA's functions.

This class is a "wrapper" for classes that describe the functions of each specific Lock-in Amplifier. It determines whether this model works with this function, and, in case of positive responce, a method of the corresponding class is called. Also, it contains test dunctions
